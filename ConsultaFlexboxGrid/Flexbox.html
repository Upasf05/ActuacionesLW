¿Qué es Flexbox?

Flexbox es un modelo de diseño incluido en CSS3 que permite distribuir y alinear elementos de forma eficiente en una página web. Fue creado para facilitar el diseño flexible y adaptable a distintos tamaños de pantalla, como celulares, tabletas y computadoras. Flexbox organiza los elementos dentro de un contenedor, permitiendo que se ajusten automáticamente según el espacio disponible.

Un contenedor Flexbox se define usando la propiedad display: flex;, y dentro de él los elementos pueden alinearse horizontal o verticalmente, con opciones para controlar el orden, el tamaño relativo, los espacios entre ellos y su envoltura. Esto elimina muchos problemas que antes se resolvían con `float` o `position`.

Propiedades esenciales

  • flex-direction: Define la dirección de los elementos (horizontal o vertical).
  • flex-wrap: Permite que los elementos se envuelvan en varias líneas si no caben.
  • justify-content: Alinea los elementos en el eje horizontal (inicio, centro, final, espacio entre ellos).
  • align-items: Alinea los elementos en el eje vertical dentro del contenedor.

Con estas propiedades es posible crear diseños modernos, centrados y responsivos de manera sencilla. Flexbox es ideal para estructuras lineales, como menús horizontales o tarjetas en una fila.
